# Changelog

All notable changes to this marketplace will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-15

### Added
- Initial release of agent-skills marketplace
- perplexity-cli plugin v1.0.0
  - Question answering with Perplexity.ai
  - Structured JSON output format
  - Multiple output formats (plain, JSON, markdown)
  - Stream response support
  - Comprehensive error handling and authentication
- Marketplace infrastructure
  - Plugin discovery and installation support
  - Documentation for users and contributors

### Changed
- Restructured repository to support Claude Code plugin marketplace
- Moved perplexity-cli skill to plugins directory
